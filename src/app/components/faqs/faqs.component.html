¿En qué consisten los componentes StandAlone? Descubre la nueva forma de crear componentes en Angular

Agregando la propiedad standalone en la creación de nuestros componentes Angular podemos hacerlos independientes de un módulo de jerarquía superior.

 ¿Los StandAlone Componentes pueden usar librerías externas? ¿Qué limitaciones tienen este tipo de componentes?

Mediante el uso de "un módulo virtual" dentro del propio componente conseguimos recoger todas las características que nos ofrece el uso de módulos en Angular pero dentro de dicho componente.

Una de estas sería el uso de librerías externas dentro del propio componente

 ¿Cómo incluimos estos componentes en nuestras rutas? ¿Podemos cargar estos componentes con Lazy Loading?

Como si de un módulo se tratase, este nuevo tipo de componentes puedes cargarse de manera "lazy" mediante el acceso a una ruta dentro de nuestra aplicación.
